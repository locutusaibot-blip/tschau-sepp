A browser-based implementation of the classic Swiss card game "Tschau Sepp" (also known as "Mau-Mau" or "Crazy Eights" in other regions).

## About the Game

Tschau Sepp is a popular Swiss card game where players take turns matching cards by rank or suit. The goal is to be the first to get rid of all your cards.

## Features

- Single-player mode (vs Computer)
- Clean, modern UI with Swiss-inspired design
- Responsive layout for mobile and desktop
- Pure JavaScript implementation - no dependencies

## How to Play

1. Open `index.html` in your browser
2. Click on cards in your hand to play them
3. Match either the rank or suit of the top card on the discard pile
4. Draw a card if you can't make a move
5. First player to empty their hand wins!

## Files

- `index.html` - Main game interface
- `game.js` - Game logic and mechanics

## Installation

Simply clone this repository and open `index.html` in any modern web browser. No build step or dependencies required!

```bash
git clone https://github.com/locutusaibot-blip/tschau-sepp.git
cd tschau-sepp
# Open index.html in your browser
```
